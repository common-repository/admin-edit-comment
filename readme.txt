== Description ==
This plugin adds another comment functionality in post/edit screen.
You can communicate with your editorial/admin team members within your WordPress screen.
In the setting page, You can enable/disable the function for each post type.
= UPDATE in version 2 =
- Display comment list on the post list screen as well.
- Show revision and status change in comment timeline.

== Installation ==
1. Upload the plugin package to the plugins directory.
2. Activate the plugin through the Plugins menu in WordPress.
